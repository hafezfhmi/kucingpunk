# kucingpunk

A website for KucingPunk NFT project

## Tech Stack

**Client:** Nextjs, Sass

## Run Locally

Clone the project

```bash
  git clone https://github.com/hafezfhmi/kucingpunk.git
```

Go to the project directory

```bash
  cd kucingpunk
```

Make sure you are running node version 12.22.0

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
